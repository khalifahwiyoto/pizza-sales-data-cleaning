# 🍕 Pizza Sales Analysis (End-to-End Project)

## 📌 Deskripsi Proyek
Proyek ini menganalisis data transaksi penjualan dari sebuah restoran pizza untuk menemukan tren penjualan harian/bulanan, kategori pizza yang paling diminati, serta ukuran pizza yang paling banyak mendatangkan keuntungan bagi bisnis.

## 🛠️ Tech Stack & Alur Kerja
1. **SQL:** Memproses query untuk mencari metrik bisnis utama seperti *Total Revenue*, *Average Order Value*, dan *Total Pizzas Sold*.
2. **Python:** Membantu dalam proses manipulasi data awal dan pemeriksaan kualitas dataset.
3. **Power BI / Tableau:** *(Tahap Selanjutnya)* Membuat dashboard interaktif untuk memvisualisasikan tren penjualan puncak dan performa menu pizza.

## 💡 Metrik Utama yang Ditemukan (SQL Insights)
Melalui analisis SQL, berhasil diidentifikasi tren hari sibuk di mana pesanan pizza melonjak drastis, serta kategori pizza (seperti Classic atau Supreme) yang menjadi kontributor terbesar bagi pendapatan restoran.
