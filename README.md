# 🍕 Pizza Sales Analysis (Data Exploration & Dashboard Preparation)

## 📌 Deskripsi Proyek
Proyek ini bertujuan untuk mengeksplorasi data transaksi penjualan dari sebuah restoran pizza. Fokus utama proyek ini adalah memahami struktur dataset, memeriksa kualitas data, dan merencanakan pembuatan dashboard interaktif.

## 🛠️ Tech Stack & Alur Kerja
1. **Python (Pandas):** Digunakan untuk melakukan inspeksi awal dataset, memeriksa tipe data, melihat statistik deskriptif, dan memastikan data siap untuk divisualisasikan.
2. **Power BI / Tableau:** *(Tahap Selanjutnya)* Membikin dashboard interaktif untuk memvisualisasikan tren penjualan puncak, performa menu pizza, dan ukuran pizza terlaris.
